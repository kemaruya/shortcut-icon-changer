# Privacy Policy — Shortcut Icon Changer

_Last updated: 2026-06-23_

## Summary

Shortcut Icon Changer does **not** collect, store, transmit, or share any personal data. The app runs entirely on your device and does not connect to the internet.

## What we collect

**Nothing.** The app has no analytics, no telemetry, no advertising, no accounts, and makes no network connections. We do not see how you use the app, and we receive no data from it.

## How the app works on your device

- It reads and writes the icon of the Windows shortcut (`.lnk`) files you choose, using standard Windows APIs.
- The icon images it applies are bundled inside the app (Microsoft Fluent UI Emoji). No download is required.
- Your settings (language, theme, default folder) and a small icon cache are stored locally on your device. They never leave your device.

## Permissions

The app runs as a full-trust Windows desktop application (`runFullTrust`) because changing a shortcut's icon requires standard Win32 file APIs. It uses this capability solely to read and modify the shortcut files you select.

## Third parties

The app does not use any third-party services, SDKs, or trackers, and shares no data with anyone.

## Children

The app is safe for all ages. Because it collects no data, it collects no data from children.

## Changes to this policy

If this policy changes, the updated version will be posted at this page.

## Contact

Questions about privacy can be raised as an issue on the project repository:
https://github.com/kemaruya/shortcut-icon-changer/issues

# プライバシー ポリシー — Shortcut Icon Changer

_最終更新日: 2026-06-23_

## 概要

Shortcut Icon Changer は、いかなる個人データも**収集・保存・送信・共有しません**。本アプリはお使いのデバイス上で完結して動作し、インターネットに接続しません。

## 収集する情報

**ありません。** 本アプリには分析（アナリティクス）・テレメトリ・広告・アカウント機能がなく、ネットワーク通信を一切行いません。利用状況が開発者に送られることはなく、開発者がアプリからデータを受け取ることもありません。

## デバイス上での動作

- お客様が選んだ Windows ショートカット（`.lnk`）ファイルのアイコンを、標準の Windows API を用いて読み書きします。
- 適用するアイコン画像はアプリ内に同梱されています（Microsoft Fluent UI Emoji）。ダウンロードは不要です。
- 設定（言語・テーマ・既定のフォルダー）と小さなアイコン キャッシュは、お使いのデバイス上にのみ保存されます。デバイス外に送信されることはありません。

## アクセス許可

ショートカットのアイコン変更には標準の Win32 ファイル API が必要なため、本アプリは完全信頼のデスクトップ アプリ（`runFullTrust`）として動作します。この権限は、お客様が選択したショートカット ファイルの読み取り・変更にのみ使用します。

## 第三者

第三者のサービス・SDK・トラッカーは一切使用せず、いかなるデータも第三者と共有しません。

## お子様の利用について

本アプリは全年齢で安全にご利用いただけます。データを収集しないため、お子様のデータを収集することもありません。

## ポリシーの変更

本ポリシーを変更する場合は、更新版を本ページに掲載します。

## お問い合わせ

プライバシーに関するお問い合わせは、プロジェクト リポジトリの Issue としてお寄せください。
https://github.com/kemaruya/shortcut-icon-changer/issues
